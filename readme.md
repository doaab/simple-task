<h3>To install</h3>
<p>After download <p>
  <p>  Create ".env" file</p>
   <p> Copy ".env.example" content to ".env"</p>
  <p> import data base from "task.sql" file and on //<b> CMD</b> //</p>
  <div style="background:#ccc"> 
    <p>composer update</p>
    <p>php artisan key:generate</p>
    <p>php artisan serve</p>
    </div>
    Now You can visit "localhost:8000"
